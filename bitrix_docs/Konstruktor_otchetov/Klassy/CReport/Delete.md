Delete
======

```
int Delete($ID);
Копировать
```

Удаляет существующий отчёт. Метод статический.

#### Параметры функции

| Параметр | Описание |
| --- | --- |
| *$ID* | Идентификатор удаляемого отчёта. |

#### Возвращаемое значение

Возвращает `true` в случае успешного удаления или `false` в случае неудачи.
