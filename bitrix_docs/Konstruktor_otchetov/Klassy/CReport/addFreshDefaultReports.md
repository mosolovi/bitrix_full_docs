addFreshDefaultReports (с версии 14.1.1)

addFreshDefaultReports
======================

```
addFreshDefaultReports($vReports, $ownerId);
Копировать
```

Добавляет отчёты по умолчанию. Метод статический.

#### Параметры функции

| Параметр | Описание |
| --- | --- |
| *$vReports* | Массив с полной информацией о добавляемых отчётах. |
| *$ownerId* | Идентификатор владельца. Как правило, содержит название модуля, к которому относятся отчёты, и наименование базовой сущности (если в модуле можно формировать отчёты на основе разных сущностей). |
