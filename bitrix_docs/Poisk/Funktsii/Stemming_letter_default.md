Stemming\_letter\_default
=========================

```
string
stemming_letter_default(
);Копировать
```

Функция возвращает алфавит по умолчанию.

#### Возвращаемое значение

Функция возвращает строку содержащую все символы латинского алфавита, а также цифры.
