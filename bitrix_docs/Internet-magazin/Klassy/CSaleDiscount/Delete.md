Delete
======

```
bool
CSaleDiscount::Delete(
	int ID
);Копировать
```

Метод удаляет скидку с кодом ID. Нестатический метод.

#### Параметры вызова

| Параметр | Описание |
| --- | --- |
| ID | Код скидки. |

#### Возвращаемые значения

Возвращается *true* в случае успешного удаления и *false* - в противном случае.
