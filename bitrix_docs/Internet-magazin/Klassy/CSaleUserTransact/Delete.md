Delete
======

```
bool
CSaleUserTransact::Delete(
	int ID
);Копировать
```

Метод удаляет транзакцию с кодом ID. Нестатический метод.

#### Параметры вызова

| Параметр | Описание |
| --- | --- |
| ID | Номер транзакции. |

#### Возвращаемые значения

Метод возвращает *true* в случае успешного удаления и *false* - в противном случае.
