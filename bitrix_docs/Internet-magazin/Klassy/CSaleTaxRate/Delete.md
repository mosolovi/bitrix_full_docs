Delete
======

```
bool
CSaleTaxRate::Delete(
	int ID
);Копировать
```

Метод удаляет ставку налога с кодом ID. Нестатический метод.

#### Параметры вызова

| Параметр | Описание |
| --- | --- |
| ID | Код ставки налога. |

#### Возвращаемые значения

Возвращается *true* в случае успешного удаления и *false* в противном случае.
