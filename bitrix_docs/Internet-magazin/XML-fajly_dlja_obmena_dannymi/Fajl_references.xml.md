Файл references.xml
===================

Включить вкладки

Описание

Пример файла

### Описание

В файле **references.xml** хранится информация о пользовательских справочниках.

| Элемент XML | Элемент Владелец XML | Описание |
| --- | --- | --- |
| <КоммерческаяИнформация> | Нет | Главный элемент XML, которому подчинены все остальные. Хранит атрибуты с информацией о пространстве имен, версии схемы («ВерсияСхемы»), даты формирования («ДатаФормирования»), а также служебную информацию. |
|  | | |
| <ПользовательскиеСправочники> | <КоммерческаяИнформация> | Элемент содержит информацию о пользовательских справочниках. У него может быть один атрибут «СодержитТолькоИзменения», который в новой схеме обмена не используется, т.к. он бесполезен при пакетном обмене. |
| <Справочник> | <ПользовательскиеСправочники> | Элемент содержит описание пользовательского справочника. |
| <Ид> | <Справочник> | Идентификатор справочника. |
| <Наименование> | <Справочник> | Наименование справочника. |
| <Реквизиты> | <Справочник> | Элемент содержит информацию о реквизитах справочника. |
| <Реквизит> | <Реквизиты> | Элемент содержит описание реквизита. |
| <Ид> | <Реквизит> | Идентификатор реквизита. |
| <Наименование> | <Реквизит> | Наименование реквизита. |
| <ТипЗначений> | <Реквизит> | Тип значения реквизита. |
| <ЭлементыСправочника> | <Справочник> | Элемент содержит информацию об элементах справочника. |
| <ЭлементСправочника> | <ЭлементыСправочника> | Элемент с информацией об элемента справочника. |
| <Ид> | <ЭлементыСправочника> | Идентификатор элемента справочника. |
| <НомерВерсии> | <ЭлементыСправочника> | Номер версии элемента справочника. |
| <ЗначенияРеквизитов> | <ЭлементыСправочника> | Элемент содержит информацию о реквизитах элемента справочника. |
| <ЗначениеРеквизита> | <ЗначенияРеквизитов> | Элемент содержит описание реквизита элемента справочника. |
| <Наименование> | <ЗначениеРеквизита> | Наименование реквизита. |
| <Значение> | <ЗначениеРеквизита> | Значение реквизита. |
