Delete
======

```
CPullWatch::Delete(
	$userId,
	$tag
);Копировать
```

Удаление подписки. Метод статический.

#### Параметры

| Параметр | Описание | С версии |
| --- | --- | --- |
| userId | Пользователь которого отписываем. |  |
| tag | Идентификатор подписки (если не указать, пользователь будет отписан от всех подписок). |  |
