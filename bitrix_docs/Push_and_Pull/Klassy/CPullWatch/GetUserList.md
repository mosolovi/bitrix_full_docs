GetUserList
===========

```
CPullWatch::GetUserList(
	$tag
);Копировать
```

Метод возвращает массив пользователей, которые подписаны на TAG. Метод статический.

#### Параметры функции

| Параметр | Описание |
| --- | --- |
| TAG | Тег, по которому производится выборка пользователей. |
