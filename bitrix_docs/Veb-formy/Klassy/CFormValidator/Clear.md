Clear
=====

```
CFormValidator::Clear(
	int FIELD_ID,
)Копировать
```

Удаляет все валидаторы, назначенные данному вопросу/полю. Метод нестатический.

#### Параметры метода

| Параметр | Описание |
| --- | --- |
| *FIELD\_ID* | ID поля. |
