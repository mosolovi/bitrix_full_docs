CSOAPFault
==========

Включить вкладки

Описание

Пример использования

### Описание

#### Члены класса

| Параметр | Описание | С версии |
| --- | --- | --- |
| *$FaultCode* | Параметр, содержащий код ошибки SOAP. | 8.0.2 |
| *$FaultString* | Параметр, содержащий строку, описывающую суть ошибки SOAP в доступном пользователю виде. | 8.0.2 |

#### Конструктор класса CSOAPFault

#### Особенность использования

Обычно экземпляр этого класса передается в метод
**ShowResponse** SOAP сервера, а тот передает **CSOAPFault**в **SOAPResponse::payload**, который различает тип ответа
*Value*: ошибка ли это (экземпляр класса **CSOAPFault**),
или же это обычный тип данных.

Таким образом, ошибки SOAP сервера, представленные классом
**CSOAPFault**, компилируются в ответ отдельным способом. Все
остальные типы проходят либо строгое кодирование согласно описанным типам
веб-сервиса, либо легкое - с помощью
**CXMLCreator::encodeValueLight**.
