addBodyAttribute
================

```
void
CSOAPRequest::addBodyAttribute( 
	string name, 
	string value
)Копировать
```

Метод добавляет атрибут к тегу **body** SOAP запроса. Нестатический метод.

#### Параметры метода

| Параметр | Описание |
| --- | --- |
| *name* | Название атрибута. |
| *value* | Значение атрибута. |
