BX.Currency.getCurrencyFormat

BX.Currency.getCurrencyFormat
=============================

```
BX.Currency.getCurrencyFormat(
	currency
);Копировать
```

Функция возвращает формат валюты.

#### Параметры функции

| Параметр | Описание | Тип |
| --- | --- | --- |
| currency | Код валюты. | string |

#### Возвращаемое значение

Возвращает `object` (если найден формат) или `false`.
