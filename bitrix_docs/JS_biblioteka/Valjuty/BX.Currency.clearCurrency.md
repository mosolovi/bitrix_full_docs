BX.Currency.clearCurrency
=========================

```
BX.Currency.clearCurrency(
	currency
);Копировать
```

Функция удаляет формат (если он есть).

#### Параметры функции

| Параметр | Описание | Тип |
| --- | --- | --- |
| currency | Код валюты. | string |

#### Возвращаемое значение

Нет.
