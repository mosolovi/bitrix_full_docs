BX.localStorage.get
===================

```
Object|String
BX.localStorage.get(
	String key
);Копировать
```

Функция позволяет получить значения элемента хранилища. Если элемента хранилища нет или его время жизни истекло, то вернется *null*.

#### Параметры функции

| Параметр | Описание |
| --- | --- |
| String key | Ключ элемента хранилища |
