BX.toggle
=========

```
mixed
BX.toggle(
	mixed param
	array values
);Копировать
```

Функция присваивает параметру **param** следующее значение из списка, задаваемого **values**. Если **param** ссылочного типа, то значение будет присвоено по ссылке.
