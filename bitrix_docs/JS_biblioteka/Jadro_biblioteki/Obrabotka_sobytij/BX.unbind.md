BX.unbind
=========

```
void
BX.unbind(
	DOMNode node, 
	string eventName,
	Function callback
);Копировать
```

Функция снимает обработчик **handler** события **event** элемента **el**.

#### Параметры функции

| Параметр | Описание |
| --- | --- |
| node | Целевой узел события. |
| eventName | Имя события, например, **click**. |
| callback | обработчик события. |
