BX.setOpacity
=============

```
BX.setOpacity(
	DOMNode node,
	integer percentage
);Копировать
```

Метод устанавливает уровень прозрачности элемента веб-страницы.

#### Параметры функции

| Параметр | Описание |
| --- | --- |
| DOMNode node | Нода |
| integer percentage | Процент прозрачности |
