BX.cleanNode
============

```
DOMNode|null 
BX.cleanNode(
	DOMNode node[,
	bool bSuicide]
);Копировать
```

Функция очищает узел DOM-структуры от всех дочерних элементов. Если вторым параметром передать **true**, сам узел также будет удален.
