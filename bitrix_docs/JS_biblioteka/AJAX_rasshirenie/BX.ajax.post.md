BX.ajax.post
============

```
XMLHttpRequest
BX.ajax.post(
	string url,
	string|object data,
	function callback
);Копировать
```

Простая отправка POST-запроса и передача результата обработчику **callback**.

#### Параметры функции

| Параметр | Описание |
| --- | --- |
| url | Адрес |
| data | Строка или ассоциативный массив POST-данных запроса |
| callback | Обработчик |
