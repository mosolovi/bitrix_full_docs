BX.ajax.insertToNode
====================

```
XMLHttpRequest 
BX.ajax.insertToNode(
	string url,
	string|DOMNode node
);Копировать
```

Функция запрашивает url и вставляет результат в контейнер node.

#### Параметры функции

| Параметр | Описание |
| --- | --- |
| url | Адрес |
| node | Если node – строка, то параметр будет интерпретирован как идентификатор контейнера. |
